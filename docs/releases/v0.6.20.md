# Release Notes — v0.6.20

**Date:** 2025-10-01
**Repository:** medagent-core

## Highlights

This release focuses on stability and developer-experience improvements
to the `medagent` core.

## Changes

### Added
- `drug` module: added configurable retry strategy with jitter
- New `--verbose` flag for detailed audit trace output
- Unit tests for health boundary conditions

### Fixed
- Off-by-one in pagination logic when result set is empty
- Memory leak in long-running drug worker loop
- Incorrect timezone handling in audit timestamps

### Improved
- Reduced average audit latency by ~12% via connection pooling
- Cleaner error messages for health validation failures

## Upgrade Notes

No breaking changes. Drop-in upgrade from the previous version.

```bash
pip install --upgrade medagent-core
```
