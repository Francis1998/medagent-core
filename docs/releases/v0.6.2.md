# Release Notes — v0.6.2

**Date:** 2024-04-08
**Repository:** medagent-core

## Highlights

This release focuses on stability and developer-experience improvements
to the `medagent` core.

## Changes

### Added
- `health` module: added configurable retry strategy with jitter
- New `--verbose` flag for detailed clinical trace output
- Unit tests for drug boundary conditions

### Fixed
- Off-by-one in pagination logic when result set is empty
- Memory leak in long-running health worker loop
- Incorrect timezone handling in audit timestamps

### Improved
- Reduced average clinical latency by ~12% via connection pooling
- Cleaner error messages for drug validation failures

## Upgrade Notes

No breaking changes. Drop-in upgrade from the previous version.

```bash
pip install --upgrade medagent-core
```
